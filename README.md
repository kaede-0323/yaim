# yaim
yaim(Yet Another make IMage) is an utility that takes screenshot of your desktop.
It's meant to overcome shortcomings of maim and parforms better in several ways.

## Features
- Takes screenshots of your desktop, and saves it in png format, or raw file. Also, it can stream to stdout.
- Takes screenshots of predetarminated regions or windows.
- Allows a user to select a region, or window, before taking a screenshot on the fly.
